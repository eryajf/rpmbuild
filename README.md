## 项目说明

这里收集整理的是生产实践中积累的RPM包构建spec文件，你可以直接基于此构建出符合自己实际需求的RPM包。

我已经构建好了openresty的基础安装包，你可以直接在 [RPMS/x86_64](./RPMS/x86_64/) 目录中找到，并可直接下载用于生产。
